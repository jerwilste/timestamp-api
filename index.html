Timestamp-API

    User Stories
  • I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 01, 2016). Entered time MUST be in 'Month DD YYYY' format.
  • If it does, it returns both the Unix timestamp and the natural language form of that date.
  • If it does not contain a date or Unix timestamp, it returns null for those properties.

Example Usage:

  • https://timestamp-jerwilste.herokuapp.com/may 04 2015
  • https://timestamp-jerwilste.herokuapp.com/October 14 2016
  • https://timestamp-jerwilste.herokuapp.com/111323253

Example Output:

  • {"natural":"May 04 2015","unix":"1430697600000"}
  • {"natural":"October 14 2016","unix":"1476403200000"}
  • {"natural":"January 02 1970","unix":111323253}
